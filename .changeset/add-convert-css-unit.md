Add `convertCssUnit` and `createCssUnitConverter` for general CSS length unit conversion. Supports `px`, `rem`, `em`, `vw`, `vh`, `vmin`, `vmax`, `lh`, `rlh`, `ch`, `%`, and absolute units. Browser auto-detect for rootFontSize and viewport when options omitted. Throws when required context is missing.

# ThemeMap <MDXBadges of={ThemeMapStoriesMeta} />

Record mapping theme keys to their values. Each value can be a single string or `readonly string[]` (e.g. multiple CSS classes). Used by all ThemeStore factories via the `themeMap` option.

## Value types

- **String value** – One string per theme (most common case).
- **Array value** – `readonly string[]` for multiple tokens (e.g. multiple CSS classes). `classNameThemeStore` applies all; `dataAttributeThemeStore` uses first only.
- **Mixed** – A `themeMap` can mix string and `string[]` values.

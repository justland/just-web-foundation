# @just-web/toolkits/react

`@just-web/toolkits/react` is a subpath of **@just-web/toolkits** that provides React specific utilities.

## Key exports

- `useAttribute` – Subscribe to element attributes
- `useThemeByClassName` – Theme from class names
- `useThemeByDataAttribute` – Theme from data attributes
- `useThemeByLocalStorage` – Theme from localStorage
- `useThemeStores` – Compose multiple theme stores
- `createThemeHook` – Factory for theme hooks

```tsx
import { useAttribute } from '@just-web/toolkits/react'
```
